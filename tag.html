ul/ol 태그

    ul태그는 순서가 없는 태그

  • disc 속성은 기본값
  • square 속성은 사각형
  • circle 원형
  • none 값이 없음

    ol태그는 순서가 있는 태그

  1. type 1
  2. type A
  3. type a
  4. type I
  5. type i

html 시맨틱 태그

header 태그는 맨 위(메뉴라고 지정하는 보통 태그)
aside는 사이드에 순서를 만드는 태그
article는 글자만 있는 태그
section는 영역 태그
footer는 하단 태그
